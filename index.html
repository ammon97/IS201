CAPM Calculator

Calculate the Expected Return using the Capital Asset Pricing Model (CAPM).

How Investors Use CAPM

If the expected return from CAPM is higher than the actual or required return, the investment might be undervalued.

If the expected return is lower, the investment might be overvalued relative to its risk.

Understanding the CAPM Model

Risk-Free Rate (Rf): Acts as the baseline return you’d expect without taking any risk, such as from a government bond.

Beta (β): Measures the asset's volatility compared to the market.

  • Beta = 1: The asset's price moves with the market.
  • Beta > 1: The asset is more volatile than the market.
  • Beta < 1: The asset is less volatile than the market.

Market Risk Premium (Rm): Represents the extra return investors expect for choosing a riskier asset over a risk-free asset.
